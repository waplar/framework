# Security Policy

Please do not publicly disclose security-related issues, [see below](#reporting-a-vulnerability).

## Supported Versions

There will be no security support for any version at this time because it has not been officially released yet.

## Reporting a Vulnerability

If you discover a security vulnerability within Waplar, please email Kaneki Yuto at kaneki.yuto.404@gmail.com. All security vulnerabilities will be promptly addressed.
